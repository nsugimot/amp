AMP について

amp-analytics

概要

アクセス解析を行う。

属性

config

設定値を、子要素の JSON ではなく、これで指定した URL から読み込む。

制限: https URL のみ

例

<amp-analytics type="googleanalytics">
  <script type="application/json">
    {
      "vars": {
        "account": "UA-74786367-1"
      },
      "triggers": {
        "trackPageview": {
          "on": "visible",
          "request": "pageview"
        }
      }
    }
  </script>
</amp-analytics>

Google Analytics での利用方法は Adding Analytics to your AMP pages を参照。
